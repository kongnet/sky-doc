01

什么是 Skybase？

Skybase 是基于 Koa2 构建的企业级 Node.js Web 框架，提供完整的 MVC 架构、API 管理和数据统计能力。

🧅

洋葱中间件

基于 Koa2 的洋葱模型，支持请求前后置处理，代码解耦更灵活

📋

API 声明式

通过配置文件定义 API，自动生成文档，支持参数扫描和校验

🗄️

双数据库支持

MySQL (j2sql2 ORM) + Redis 一站式数据层，Promise 风格 API

📊

实时统计

RTS 实时统计组件，支持多时间粒度的数据埋点和聚合分析

🔧

开发工具

API 参数扫描、安全扫描、热更新，开发效率最大化

🔐

安全认证

内置 Token 验证、签名校验、SQL 注入检测等安全机制

02

目录结构

project-root/ ├── router/ // 路由(处理Controller) ├── model/ // 数据模型/API定义 ├── service/ // 业务逻辑层 ├── middleware/ // 自定义中间件 ├── config/ // 环境配置 ├── tool/ // 开发工具 ├── : var(--accenttests/ // 测试文件 ├── index.js // 入口文件 └── package.json
  • router/ - 路由处理层，接收请求，调用 service
  • model/ - 数据模型，包含 API 定义和数据库模型
  • service/ - 业务逻辑层，处理核心业务
  • middleware/ - 自定义中间件，如鉴权、日志
  • config/ - 多环境配置 (dev/test/prod)
  • tool/ - 开发工具，如代码生成器

API 定义 - 概念

Skybase 使用声明式配置定义 API，通过 model/api/ 目录下的文件定义接口规范。

API 配置项说明

字段类型说明
namestringAPI 名称（中文）
descstringAPI 描述
methodstring请求方法: get/post/all
controllerstring路由: module.method
paramobject参数定义与校验规则
tokenboolean是否需要 token 验证
needSignboolean是否需要签名校验

API 定义 - 示例

定义用户接口

03

Controller 开发

Controller 位于 router/ 目录，负责接收请求参数、调用业务逻辑、返回响应。

基本结构

// router/user.js /* global $ db redis */ module.exports = { async info(ctx) { // 获取参数 const { userId } = ctx.checkedData.data // 业务逻辑 const user = await db.user.find(userId) // 返回响应 if (!user) { ctx.throw(404, '用户不存在') } ctx.ok(user) }, async update(ctx) { const { userId, nickname, avatar } = ctx.checkedData.data await db.user.update(userId, { nickname, avatar }) ctx.ok('更新成功') } }

响应方法

  • ctx.ok(data) - 成功响应
  • ctx.throw(status, msg) - 抛出错误
  • ctx.checkedData.data - 已校验的参数
  • ctx.$token - 当前登录用户信息

全局变量

  • $ - Meeko 工具库
  • db - MySQL 连接
  • redis - Redis 连接
03

中间件开发

中间件位于 middleware/ 目录，遵循 Koa2 洋葱模型，可在请求前后执行逻辑。

基本结构

// middleware/auth.js /* global $ redis */ module.exports = async (ctx, next) => { // 前置处理 const token = ctx.headers.authorization if (!token) { ctx.throw(401, '请先登录') } // 验证 token const userInfo = await redis.get(`token:${token}`) if (!userInfo) { ctx.throw(401, '登录已过期') } // 挂载用户信息 ctx.$token = JSON.parse(userInfo) // 继续执行 next && (await next()) // 后置处理 // ... }

洋葱模型执行顺序

中间件 1 (前置)
中间件 2 (前置)
Controller 业务逻辑
中间件 2 (后置)
中间件 1 (后置)

MySQL - 查询操作

基础查询

// 根据 ID 查询 const user = await db.user.find(1) // 查询单条 const user = await db.user.find({ where: { status: 1 } }) // 查询多条 const users = await db.user.findAll({ where: { type: 'vip' } }) // 条件查询 const list = await db.user.where({ age: { $gte: 18 } }).all() // 排序分页 const list = await db.user.order('created_at', 'desc') .limit(10).offset(0).all() // 统计数量 const count = await db.user.count({ where: { status: 1 }) })

运算符

$eq // 等于 (=) $ne // 不等于 (!=) $gt // 大于 (>) $gte // 大于等于 (>=) $lt // 小于 (<) $lte // 小于等于 (<=) $like // 模糊匹配 (LIKE) $in // IN 查询 $notIn // NOT IN 查询 $between // 范围查询 // 使用示例 await db.user.where({ age: { $between: [18, 30] }, name: { $like: '%张%' } }).all()

MySQL - 事务操作

事务用于保证多个数据库操作的原子性，要么全部成功，要么全部失败。

事务示例 - 用户转账

async transfer(fromId, toId, amount) { // 开始事务 const conn = await db.transaction() try { // 扣除转出账户余额 await conn.user.decr(fromId, 'balance', amount) // 添加转入账户余额 await conn.user.incr(toId, 'balance', amount) // 记录转账日志 await conn.transferLog.add({ from_id: fromId, to_id: toId, amount: amount, created_at: new Date() }) // 提交事务 await conn.commit() return true } catch (err) { // 回滚事务 await conn.rollback() throw err } }

Redis - 基础操作

String 类型

// 设置值 await redis.set(`user:${userId}`, JSON.stringify(user)) // 设置带过期时间 (秒) await redis.setex(`code:1234`, 300, '1234') // 获取值 const data = await redis.get(`user:${userId}`) // 自增/自减 await redis.incr('page:views') await redis.incrby('count', 10) await redis.decr('stock') // 批量操作 await redis.mset({ 'k1': 'v1', 'k2': 'v2' }) const vals = await redis.mget(['k1', 'k2'])

Hash 类型

// 设置 Hash await redis.hset(`user:${userId}`, { name: '张三', age: 25, email: 'test@example.com' }) // 获取 Hash const user = await redis.hgetall(`user:${userId}`) // { name: '张三', age: '25', ... } // 获取单个字段 const name = await redis.hget(`user:${userId}`, 'name') // 自增 Hash 字段 await redis.hincrby(`user:${userId}`, 'score', 10) // 删除字段 await redis.hdel(`user:${userId}`, 'field')

Redis - 实战场景

缓存用户信息

async getUserInfo(userId) { // 先查缓存 const cacheKey = `user:${userId}` const cached = await redis.get(cacheKey) if (cached) { return JSON.parse(cached) } // 缓存未命中，查数据库 const user = await db.user.find(userId) // 写入缓存 (1小时) await redis.setex(cacheKey, 3600, JSON.stringify(user)) return user }

接口限流

async rateLimit(userId, max = 100) { const key = `ratelimit:${userId}` // 限流窗口: 60秒 const window = 60 const current = await redis.incr(key) if (current === 1) { await redis.expire(key, window) } if (current > max) { ctx.throw(429, '请求过于频繁') } return true }
